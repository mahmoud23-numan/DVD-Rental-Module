# DVD Rental Odoo Module

This module implements a DVD rental management system in Odoo.

## Features

- Manage Films
- Manage Actors
- Manage Categories
- Manage Customers
- Rental Management
- Payment Tracking

## Models

- Film
- Actor
- Category
- Inventory
- Rental
- Payment
- Customer
- Address
- City
- Country
- Store
- Staff
